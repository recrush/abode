# abode

The Place Where Data Lives

## Quick Start

Start a abode instance with fs as backend.

```shell
export ABODE_ADDR=127.0.0.1:8080
export ABODE_BACKEND_TYPE=fs
export ABODE_BACKEND_FS_ROOT=/tmp/

./abode http
```
